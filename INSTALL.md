# Установка зависимостей

## Системные пакеты
Перед установкой Python-зависимостей поставьте шрифты Roboto, которые используются в шаблоне (`Roboto` и `Roboto Mono`). Для Debian/Ubuntu достаточно запустить скрипт:

```bash
./scripts/install_fonts.sh
```

Скрипт выполнит `apt-get update` и установит нужные семейства (`fonts-roboto`, `fonts-roboto-unhinted`, `fonts-roboto-mono`). В других окружениях установите эти шрифты вручную так, чтобы XeLaTeX видел их в системных шрифтах.

Также необходимы `pandoc 3.x` и `xelatex` (например, из пакета `texlive-xetex`).

## Python-зависимости
1. Создайте виртуальное окружение Python 3.11+ и активируйте его.
2. Установите зависимости проекта:
   ```bash
   python -m pip install -e .
   ```

После этого можно запускать `ruff`, `mypy`, `pytest` и рендеринг PDF.
